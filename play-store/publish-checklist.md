# Publish Checklist

- [ ] Confirm package name `com.geison.tabuada` exists in Play Console.
- [ ] Configure GitHub repository secrets.
- [ ] Add Tabuada-specific Play Store icon, feature graphic, and screenshots under `play-store/graphics/` and `play-store/screenshots/` before uploading listing images.
- [ ] Run `./gradlew test`.
- [ ] Run `./gradlew bundleRelease` or the `Android Closed Testing Release` workflow.
- [ ] Upload or refresh store listing metadata.
- [ ] Verify privacy policy URL.
- [ ] Complete Play Console app content declarations.
- [ ] Submit the closed testing release for review.
